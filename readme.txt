# Visor Compacto

Una aplicación ligera y elegante para ver, rotar, hacer zoom y gestionar imágenes.

## Características
- Carga múltiples imágenes
- Navegación entre imágenes ("Correr")
- Borrar imagen actual o todas
- Zoom (+ / –)
- Rotación (90°)
- Descarga directa
- Diseño moderno y responsive

## Cómo usar
1. Abre `index.html` en tu navegador.
2. Haz clic en **Cargar** y selecciona una o varias imágenes.
3. Usa los controles para navegar y editar.

## Compatibilidad
Funciona con: JPEG, PNG, GIF, WebP, BMP y más (según soporte del navegador).
